FILE FINDER

File Finder is the native advanced graphical search tool with an integration into Nautilus File Explorer. There is multiple search location support with threading, file masking, MIME types, modified dates, file sizes, text and binary patterns. The search results can be sorted by attributes and processed with common actions like opening with a default application, containing folder, copy/move to destination folder or trashing selection.
